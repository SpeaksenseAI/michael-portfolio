## My Contribution

FurrowVision is a long-running John Deere Precision Agriculture program (5+ years). My 1.5-year engagement covered three workstreams: model accuracy and architecture, dataset engineering, and analytics infrastructure.

**Baseline going in**: F1 of 0.44 on the hardest classification classes, manual annotation pipeline, no systematic path from 38M-image corpus to training set.

## Model & Architecture

- **Architecture search**: Evaluated SwinTransformer, MobileNetV2, and EfficientNet-B0 against real-time latency SLAs on A10 GPU clusters. SwinTransformer was eliminated — parameter count made retraining costly and latency exceeded field requirements. MobileNetV2 and EfficientNet-B0 both cleared the SLA; EfficientNet-B0 edged accuracy, MobileNetV2 won on compression headroom for potential edge deployment.
- **Result**: F1 lifted from **0.44 → 0.92** through iterative architecture evaluation combined with dataset curation below.

## Dataset Engineering

The corpus held **38M+ sensor images** across geographies, crop types, and lighting conditions. Prior training sets of 60,000+ images underperformed because they were built by class-label sampling, not semantic coverage.

- **DINOv2 embedding curation**: Computed inter-sample distances across the full corpus via FiftyOne. Sampled by embedding-distance quartile to fill underrepresented regions of the visual space — not by label count.
- **Hard-negative targeting**: Located the failure clusters (motion blur, edge ambiguity, partial occlusion) and concentrated annotation effort there, shrinking the false-negative tail without growing dataset size.
- **Outcome**: ~15,000-image training set that outperformed all prior 60,000+ image sets. Post-handoff, CVML engineers added **4,000+ images** using the same embedding workflow — no data scientist required.

## Analytics Pipeline

Parallel workstream converting model outputs into business metrics:

- **7-table schema** from raw sensor telemetry → 1Hz aggregation → acre-level dashboard summaries
- **17 system metrics** formally contracted with Systems Engineering, ensuring consistent interpretation across **300+ acre-class deployments**
- **Brazil expansion**: Integrated a second sensor hardware variant and coordinate system without restructuring the core pipeline
- **Delta Lake + Unity Catalog** on Databricks for full training-run lineage and reproducibility
