## What Klarix Does

Klarix delivers competitive intelligence as a service — per-company prospect dossiers, SWOT analyses, battlecards, and personalized outreach sequences for B2B sales and recruiting teams. Each engagement produces 100–400 research artifacts across 25–100 companies, delivered in 3–7 days.

The pipeline is the product. Manual delivery at this depth and volume isn't possible without agentic automation.

## Pipeline Architecture

The core challenge: each research stage has different latency, cost, and quality tradeoffs. A single model can't handle all of them efficiently.

Six providers are assigned by stage:

| Stage | Model | Reason |
|-------|-------|--------|
| ICP scoring | MiniMax M2.7 Highspeed | Low cost, high throughput at 950 RPM |
| Research synthesis | Gemini 3.1 Pro | Long context, multi-source summarization |
| Outreach copy | Claude 4.5 Opus | Highest output quality for written deliverables |
| Contact collection | Groq Compound + Llama 70B | Sub-second latency for structured extraction |
| Deep research | Tavily | Real-time web retrieval |
| Contact data | Apollo API | Verified email + firmographic enrichment |

Cost-aware routing allocates the cheapest model that clears quality thresholds per stage — not the best model uniformly.

## Reliability Engineering

At 128 concurrent threads and 450–950 RPM across providers, partial failures are expected. Three systems prevent data loss:

- **3-tier failover**: MiniMax → OpenRouter → Groq. Each tier activates automatically on rate-limit or API error, with no manual intervention.
- **Append-under-lock JSONL checkpointing**: Every completed record is flushed to disk immediately. A crashed run resumes from the last checkpoint — not from scratch.
- **Rate limit management**: Per-provider token budgets enforced at the thread pool level, preventing cascading failures when one provider degrades.

## Output Quality

Each run produces structured, source-cited deliverables per company:

- **Prospect dossier**: Company overview, evidence-based pain points, buying triggers, decision-maker map with contact scores, personalized outreach hooks
- **SWOT analysis**: Strengths, weaknesses, opportunities, threats with competitive context
- **Battlecard**: Talk tracks, landmines, key contacts at competitors
- **Outreach sequence**: LinkedIn + email, 3-touch cadence, objection handling, voicemail script
- **Executive one-pager**: Compressed summary for leadership review
