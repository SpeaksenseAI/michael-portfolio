## Overview

Led the complete rebuild and rebrand of the Jolt Foundation's website, transforming it into a modern, user-friendly platform that better serves the local community. The project focused on improving navigation, enhancing visual design, and optimizing the user experience across all devices to increase community engagement and resource accessibility.

## Key Features

- **Complete Site Architecture**: Rebuilt the website from ground up with improved navigation, layout, and branding to better serve community needs.
- **Mobile-First Responsive Design**: Achieved 100% mobile compatibility across all devices with optimized user experience on both desktop and mobile platforms.
- **Performance Optimization**: Reduced page load times by 40%, creating a smoother and more accessible user experience.
- **Multiple Design Options**: Created and presented four unique design concepts using wireframes and prototypes to ensure the best possible solution.

## Technologies Used

- **Figma**: For wireframing, prototyping, and design system creation.
- **Adobe XD**: For additional prototyping and design iterations.
- **Responsive Web Design**: Implemented mobile-first design principles for optimal cross-device experience.
- **User Experience Research**: Conducted user testing and feedback sessions to inform design decisions.

## Challenges and Learnings

The main challenge was balancing the organization's diverse community needs while creating a cohesive, accessible platform. This required extensive user research to understand how different community segments interacted with the site, followed by iterative design and testing. The 5-month optimization process taught valuable lessons about performance optimization and the importance of mobile-first design in community-focused platforms.

## Outcome

- **Increased Community Engagement**: Successfully provided resources to over 500 local community members
- **Improved User Engagement**: Achieved 30% increase in user engagement through better navigation and layout
- **Enhanced Performance**: 40% reduction in page load times and 100% mobile compatibility rate
- **Significant Traffic Growth**: 50% increase in website traffic post-launch, demonstrating improved user experience and accessibility
